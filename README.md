# Linktree Algedian – Pemrograman Web

## Deskripsi Project
Website ini berfungsi sebagai halaman profil yang menampilkan **foto profil**, **background**, dan beberapa **link menuju media sosial**.

## Fitur
- Tampilan halaman satu layar (single page)
- Foto profil di tengah halaman
- Background menggunakan gambar
- Tombol link ke media sosial:
  - WhatsApp
  - Instagram
  - (dan media sosial lainnya)
- Setiap tombol akan langsung membuka aplikasi / website tujuan
- Responsive (dapat dibuka di HP dan desktop)

## yang Digunakan
- **HTML** → struktur halaman
- **CSS** → desain dan layout
- **Font Awesome** → ikon media sosial
- **Git & GitHub** → version control
- **GitHub Pages** → deployment website
